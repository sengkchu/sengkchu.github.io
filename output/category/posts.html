Visualizing Earnings based On College Majors¶

In this project we will look at earnings from recent college graduates based on each major in 'recent-grads.csv'. We'll visualize the data using histograms, bar charts, and scatter plots and see if we can draw some interesting insights from it. However, the main purpose of this project is to practice some of the data visualization tools.
